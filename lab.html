🧪 Laboratório de fricção UX

Cada bloco abaixo foi feito para gerar um sinal específico no Clarity. Teste como um usuário real.

1. Clique morto (Dead click)

Elemento que parece botão, mas não responde.

Salvar preferências
No Clarity: Dashboard → Dead clicks

2. Clique de raiva (Rage click)

Este botão só funciona no 3º clique. Clique rápido várias vezes.

No Clarity: Dashboard → Rage clicks

3. Resposta lenta

Demora 3 segundos para responder. Muita gente clica de novo antes.

Veja nas gravações se o usuário clica repetidamente durante a espera.

4. Botão camuflado

A ação importante está quase invisível.

Compare no heatmap: poucos cliques nesse micro-botão.

5. Modal confuso

O X não fecha. O fechamento real é um link discreto.

Nas gravações, observe tentativas no X falso antes de achar "descartar".

6. Scroll profundo

Conteúdo longo com CTA só no final — testa até onde o usuário rola.

Linha 1: conteúdo filler para forçar rolagem.

Linha 2: a maioria desiste antes do fim.

Linha 3: continue rolando...

Linha 4: quase lá...

Linha 5: mais um pouco...

Linha 6: último trecho visível no scroll interno.

Linha 7: você chegou ao fundo!

No Clarity: Heatmaps → Scroll map desta página.
